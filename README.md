# 📊 Trabajo Práctico Integrador 2026 Bruno Corletti

Este repositorio contiene el desarrollo del Trabajo Práctico Integrador correspondiente a la materia **Introducción a la Programación Orientada a Datos**.

## Contenido

En el notebook se desarrollan las siguientes actividades:

- Conversión de una lista de datos en un DataFrame.
- Cálculo de indicadores generales.
- Clasificación de estudiantes según su situacion académica.
- Comparación del rendimiento academico entre distintos grupos de estudiantes.
- Identificación de valores máximos y mínimos.
- Generación de gráficos para visualizar los resultados.
